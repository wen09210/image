點擊時發揮作用，以id為觸發對象可自行更改名稱，content為觸發內容，content內容可自行編排

HTML

<div  data-toggle="popover" data-placement="bottom" id="popOver">Demo</div>
<div id="popOver-content" class="hidden">
	<div class="h3">
		h3h3h3h3hh33
	</div>
	<h4>h4h4hh4h4h4</h4>
</div>

JS

觸發前請先呼叫 
$("#popOver").popover({
html: true,
content: function () {
return $("#popOver-content").html();
}
})

位置data-placement="" 有四個方向，bottom，top，left，right可自行調整
